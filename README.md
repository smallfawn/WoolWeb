# 项目搭建：
# 1.前端
1).把前端文件放到网站目录(宝塔面板为网站根目录 1panel面板为网站根目录index文件下)

2).配置接口(设置反向代理/Public代理到公共api,设置/private代理到私有api,本项目共使用两个后端)

# 2.后端(私有)
1).把后端文件放到任意目录下(确保环境存在node -v)

2).安装基本依赖(cd到项目目录)
``````shell
npm install
``````
``````shell
npm install pm2 -g
``````
``````shell
pm2 start Api.js
``````
